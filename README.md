# Music Streaming Library (Java OOP)

This project demonstrates basic Object-Oriented Programming concepts in Java.

## Features
- Classes: Artist, Song, Playlist
- Constructors, attributes, methods
- Getter and Setter methods
- Object creation in main method
- Console output
- Object comparison using equals()

## Technologies
- Java
- OOP principles
